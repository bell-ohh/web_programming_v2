Metadata is extra information about a webpage that allows a computer to understand it. People may not see it. In HTML, !doctype html tells a browser how modern the page is, and htmlis the main wrapper. In the head structure, title names the page tab in the browser, and meta tags add a short description for search engines.


Metadata also helps the page fit on phones, and make sure the text is showing correctly. Basically, metadata helps the page show up better and work online.
The body consists of the actual content.
